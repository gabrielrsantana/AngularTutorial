Módulos no Angular

Pré configuração

Para evitar erros de compilação por causa do modo restrito, o recomendado é desabilitá lo no projeto.

Para desabilitar o modo restrito strict no Angular, edite o arquivo tsconfig json localizado na raiz do projeto, e altere a seguinte linha de

1. strict true Para strict false

Pronto, feito isso o modo restrito strict será desabilitado e nenhum erro de compilação por causa dele será gerado

Criando um projeto

  1. ng new ola-mundo
  2. cd ola-mundo
  3. ng serve
  4. Remover componentes de teste em app.componentes.spec.ts
  5. Apagar todo o conteúdo do arquivo app.componente.html
  6. Digitar algo no app.component.html, como Olá mundo!
  7. Abra uma nova janela cmd , entre no direitório do projeto calculadora. (você ficará utilizando 2 prompts, 1 para que a aplicação fique em execução e outro pra executar alguns comandos necessários.)
  8. Va no navegador e digite: http://localhost:4200/

Criando o Módulo

No contexto do Angular um módulo é a forma de agrupar componentes, diretivas, pipes e serviços que são relacionados. Esse conjunto de módulo são “encaixados” para formar a aplicação, isso mesmo como se fosse peças de Lego. Um módulo pode esconder ou exportar um componente, diretiva, serviço ou pipe

Todo app angular tem pelo menos um módulo

para criar um módulo dentro com projeto anterior, vá para o prompt dentro da pasta do projeto e digite: ng g module ola-mundo.

deverá ser criada uma pasta dentro de src ->app ->ola-mundo
